Projects

SpamSleuth

An application designed to help older individuals recognize and protect themselves from online scams. Built using React, Tailwind, Flask, Tensorflow, Scikit-learn, and an OpenAI API. The project won the Congressional App Challenge 2024, earning the opportunity to be presented at the US Capitol's annual House Of Code in April 2025.

Learning Enrichment Engine

The Learning Enrichment Engine is a project which increases access for free learning opportunities to students across Illinois, developed by IMSA students in collaboration with the president of IMSA and the Illinois Association of Gifted Children. I have been working on this project for over a year, and have been serving as the co-lead developer and leader for half a year. It is consistently being updated and built using PHP, Bootstrap, and MySQL.

Destination Aurora & IBCez.ai

I worked on both of these projects during my summer internship at APS Data Technologies. Destination Aurora is an app commissioned by the City of Aurora. It is an all-in-one guide which allows you to book movie tickets, dinner reservations, and more all from one location.

I spent the majority of my internship working on IBCez.ai, an app to make it easier for construction workers to find building codes. Both apps were built using React Native and Firebase, with IBCez.ai using a tensorflow model. Destination Aurora is now downloadable on the App and Google Play Stores!
